할 일 · Google Sheets

데모 데이터 표시 중
담당자 색상 순서 (쉼표로 구분)
할 일은 구글 시트에서 가져와 읽기 전용으로 표시합니다(편집은 모바일 구글 시트 앱에서). 첫 줄을 머리글로 두고 열 이름을 할일 · 담당자 · 우선순위 · 완료 · 완료일 로 만드세요(순서 무관, 일부 생략 가능).

연결 방법 (권장) — 구글 시트에서 파일 → 공유 → 웹에 게시 → 형식 쉼표로 구분된 값(.csv) 선택 → 게시 → 나온 링크(…/pub?output=csv)를 위에 붙여넣고 불러오기. (별도 키·로그인 불필요)

⚠️ 일반 편집 링크를 넣었을 때 NetworkError가 나면 시트가 완전 공개가 아니라는 뜻입니다 — 위의 “웹에 게시 → CSV” 방식을 쓰면 해결됩니다.

우선순위 칸이 높음/⭐/p1 이면 별로, 완료 칸이 완료/done/y 이면 완료 처리되어 아래로 내려가 흐려집니다. 완료일(날짜)을 적으면 매주 일요일 자정에 지난 완료 항목이 자동 정리됩니다.

캘린더 · Google Calendar

데모 일정 표시 중
일정은 Google 캘린더에서 가져와 표시하고, 한국 공휴일은 같은 키로 자동으로 함께 불러와 빨간색으로 표시합니다. 한 캘린더에 모아 쓰고 싶으면, 제목 앞에 [이름] 대괄호를 붙이면 담당자별 색으로 구분됩니다 — 예: [시윤] 급여일, [하나] 생일, [공동] 차량 할부. 화면에는 대괄호를 뗼 제목만 보입니다.

연결 방법 — 1) Google Cloud Console에서 프로젝트 생성 → Calendar API 사용 설정 → 사용자 인증 정보 → API 키 생성. 2) 휴대폰/웹 Google 캘린더에서 해당 캘린더 설정 → 액세스 권한 → ‘공개 사용 설정’. 3) 같은 화면의 캘린더 통합 → 캘린더 ID 를 복사. 4) 위에 API 키와 캘린더 ID를 넣고 불러오기. (OAuth·로그인 불필요)

⚠️ 공개로 설정해야 키만으로 읽힙니다. 가족 일정 공개가 부담되면 일정 전용 캘린더를 따로 만들어 공개하는 방법을 권장합니다.
반복 일정 (규칙 · Google Sheets)
시트 미연결
고정 날짜가 아닌 규칙 반복(예: 매월 마지막 평일=월급날)을 위한 시트입니다. 머리글을 제목 · 규칙 으로 만들고 규칙을 한국어로 적으면 매달 자동 계산됩니다. 제목에 [이름]을 붙이면 담당자 색으로 표시됩니다.

규칙 예시
• 매월 마지막 평일 (주말 제외 그 달 마지막 근무일 — 월급날)
• 매월 첫 평일 / 매월 말일
• 매월 25일 (없는 달은 말일로)
• 매월 마지막 금요일 / 매월 둘째 토요일
• 매주 월요일

예: 제목 [하나] 월급날, 규칙 매월 마지막 평일

생일 · 기념일

시트 미연결
생일·기념일을 구글 시트에서 읽어와 캘린더에 매년 🎂(생일) · 🎉(기념일) 로 표시합니다. 편집은 모바일 구글 시트 앱에서 하면 30분마다 자동 반영됩니다.

시트 형식 — 첫 줄을 머리글 유형 · 이름 · 날짜 · 양력/음력 로 만드세요. 예: 생일, 시윤, 1996.03.13, 양력 / 기념일, ❤️, 2014.06.06, 양력. 연도를 적으면 기념일이 N주년으로 표시되고, 음력이면 매년 양력 날짜가 자동 계산됩니다(한국 음력, 1900~2100년).

연결 — 구글 시트에서 파일 → 공유 → 웹에 게시 → CSV 링크를 위에 붙여넣고 불러오기. (별도 키·로그인 불필요)
